# 꼬마온라인
## 팀원 소개
### 개발
이승혁 Photon 을 사용하여 온라인 멀티플레이 기능, 플레이어 애니메이션과 전체적인 플레이의 흐름을 구현. <br>
김범주 UI 및 연출 개발
### 기획
김건범
## 게임설명
5라운드로 구성된 온라인 캐주얼 게임입니다.
